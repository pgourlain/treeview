un autre TreeView....
Connaissez-vous le Virtual Treeview de Mike Lischke ?, les Delphistes certainement. ce Treeview est tout simplement excellent !. Ce composant LGPL est très largement utilisé, pour l'exemple Borland l'utilise très largement au sein de l'ide Delphi 2005

Malheureusement l'auteur indique sur son forum  qu'il ne portera pas son treeview en .Net 
Quote:
One more question: Will the new version of VTV be compatible with VCL for .NET (in Delphi 2005)?
No, this will not be the case. .NET is a totally different beast than Win32.

Mike

j'ai donc décidé de me lancer dans ce développement,  très inspiré du treeview de Mike Lischke, mais en C#.

principales fonctionnalités :

  • surligné un chemin 
  • support des cases à cochées
  • support des colonnes ou mode grille, avec personalisation de la couleur de fond
  • support d'un header de colonne
  • support des images et image d'état (ImageState) sur les noeuds
  • support du drag'n drop dans le header
  • support du drag'n drop dans le treeview 

fonctionnalités absentes  :

  • pas de designer pour visual studio
  • pas d'aide
  • un seul exemple, celui avec lequel j'ai fait les aperçus
  • ...

- drag'n drop l'image de drag est rouge car "Node 2" ne peut être lacher sur lui-même
cette image en drag est réalisé avec une NativeWindow  sur laquelle j'utilise l'api SetLayeredWindowAttributes, avec une valeur alpha à 170, 255 étant opaque 

      CreateParams = new CreateParams ();
      cp.X = x; //position de la fenêtre
      cp.Y = y;
      cp.Height = h;
      cp.Width = w;
      WindowStyles style = (WindowStyles.WS_POPUP );
      cp.Style = ( int )style;
      cp.Parent = FTree.Handle ;
      if (alpha < 255 )cp.ExStyle |= ( int )WindowExStyles.WS_EX_LAYERED ;
      this .CreateHandle (cp) ;
      if (alpha < 255 )NativeMethods.SetLayeredWindowAttributes ( this .Handle , 0 , alpha, 0x02 ) ;


pour rendre rougeâtre l'image en drag j'utilise la classe System.Drawing.Imaging.ImageAttributes et System.Drawing.Imaging.ColorMatrix

ImageAttributes attr = new ImageAttributes();
ColorMatrix matrix = new ColorMatrix();
matrix.Matrix00 = 1.00f; // Red
matrix.Matrix11 = 1.00f; // Green
matrix.Matrix22 = 1.00f; // Blue
//permet de rendre transparent le bitmap, par rapport à la couleur de fond
matrix.Matrix33 = 0.5f; // alpha
matrix.Matrix44 = 1.0f; // w
attr.SetColorMatrix(matrix);
